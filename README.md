# Pacman
Jogo de pacman

Para rodar o jogo, basta baixar o arquivo Pacman-5.0.jar e rodar executando o seguinte comando:

java -jar Pacman-5.0.jar

Ou, se estiver no windows, pode dar dois cliques no arquivo Pacman-5.0.jar.
